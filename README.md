# Tailwind PWA Test

### About

Testing Tailwind PWA Boilerplate

### Development

First time setup: `npm i`

Generate Assets: `npm run generate-assets` with icon.svg saved under /src/img

Development: `npm run dev`

Production: `npm run prod`

Serve: `npm run serve`
